Data-Analysis-hot-spots
Analise de dados de focos de calor registrados pelo Instituto de Pesquisas Espaciais - INPE do Brasil nos meses de setembro e outubro de 2018.

Utilizando um mapa de calor e as informações cedidas pelo portal de queimadas do Inpe onde apresenta os focos de calor registrados pelo Instituto de Pesquisas Espaciais - INPE do Brasil nos meses de setembro e outubro de 2018. Projeto elaborado para a materia de Graduação de Algoritmos na FATEC de São José dos Campos.

Fontes:
Portal de Queimadas/BD Queimadas do Inpe: http://www.inpe.br/queimadas/bdqueimadas

usada para criar o mapa: https://media.readthedocs.org/pdf/folium/latest/folium.pdf
Antes de executar
É necessario fazer a instalação das bibliotecas Pandas e Folium através dos comandos: pip install pandas
pip install folium
